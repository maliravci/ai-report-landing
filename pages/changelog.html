Changelog

Track every update to ReportDrafter — new features, improvements, and fixes.

v2.4.0

February 25, 2026

Added Batch Report Generation

Generate up to 50 reports simultaneously by uploading multiple source document sets against a single template. Progress tracking and individual download links are shown in a new batch dashboard.

Added Field Locking

Lock individual fields after review so they are preserved across re-extraction runs. Locked fields display a lock icon and are skipped during subsequent AI processing.

Improved Confidence Score Accuracy

Updated the confidence scoring model to better calibrate scores for tabular data, reducing overconfident ratings on ambiguous numeric fields by approximately 15%.

Fixed Export Formatting on .xlsx

Resolved an issue where merged cells in Excel templates could cause misaligned field placement in exported reports.

v2.3.0

January 18, 2026

Added 8 New Interface Languages

ReportDrafter is now available in 21 languages, adding support for Korean, Thai, Vietnamese, Polish, Czech, Romanian, Hungarian, and Greek.

Improved Upload Performance

Large document uploads (>50 MB) now use chunked transfers with automatic retry, reducing failure rates by 80% on unstable connections.

Fixed Mobile Navigation

Fixed a bug where the report field list could overflow the viewport on narrow Android screens.

v2.2.0

December 5, 2025

Added API Access (Beta)

Launched the ReportDrafter REST API in public beta. Teams can now programmatically upload documents, trigger extraction, retrieve results, and download generated reports. See the API Reference for details.

Added Workspace Roles

Introduced Admin, Editor, and Viewer roles per workspace. Admins can manage members and billing; Editors can create and edit reports; Viewers have read-only access.

Improved Re-extraction Speed

Selective re-extraction for individual fields is now 3× faster thanks to a new caching layer that preserves unchanged document context.

v2.1.0

October 22, 2025

Added Confidence History

View a timeline of confidence score changes for any field across extraction runs. Useful for auditing and demonstrating review diligence.

Improved Template Detection

The template parser now supports nested tables, repeated sections, and conditional blocks in .docx and .xlsx files.

Fixed PDF Source Encoding

Fixed a character encoding issue that affected extraction accuracy on PDFs with non-Latin scripts.

v2.0.0

September 1, 2025

Added Cross-Platform Launch

ReportDrafter is now available on iOS and Android in addition to web. All platforms share the same extraction engine and field review workflow.

Added Confidence Scoring

Every extracted field now includes a confidence score (0–100%) with color-coded indicators: green (≥90%), amber (70–89%), and red (<70%).

Improved Complete UI Redesign

Rebuilt the interface with a cleaner layout, improved typography, and better information density for reviewing large reports.
