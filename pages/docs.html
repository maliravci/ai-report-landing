Troubleshooting

Extraction returns low confidence on all fields

This usually means the source document format isn't matching well with the template structure. Try using clearer field labels in your template, or ensure the source document contains the expected data in a parseable format (not scanned images).

Upload fails for large files

The maximum file size is 100 MB per document. For larger files, consider splitting them into sections. If uploads fail on smaller files, check your network connection â€” the platform uses chunked uploads with automatic retry.

Exported report has missing formatting

Complex formatting (conditional formatting, macros, VBA) may not be preserved in all cases. Stick to standard cell formatting, fonts, and merged cells for best results.
